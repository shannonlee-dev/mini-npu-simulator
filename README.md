# Mini NPU Simulator

## 1. 프로젝트 개요

이 프로젝트는 입력 패턴과 필터를 위치별로 곱한 뒤 모두 더하는 MAC(Multiply-Accumulate) 연산을 직접 구현하여,
패턴이 Cross에 가까운지 X에 가까운지 판별하는 Python 콘솔 애플리케이션이다.

프로그램은 두 가지 모드를 제공한다.

1. 3×3 필터와 패턴을 직접 입력받아 판정하는 사용자 입력 모드
2. `data.json`을 읽어 여러 크기의 패턴을 일괄 판정하는 JSON 분석 모드

또한 라벨 정규화, epsilon 기반 동점 처리, 크기별 성능 분석, 실패 원인 분석까지 포함한다.

## 2. 실행 환경

- Language: Python 3.11
- Library: Python 표준 라이브러리만 사용
- Main file: `main.py`
- Data file: `data.json`

## 6. 핵심 구현 요약

- 라벨 정규화: `+`는 `Cross`, `x`는 `X`로 변환
- MAC 연산: 외부 라이브러리 없이 중첩 반복문으로 구현
- 동점 처리: `abs(score_a - score_b) < 1e-9`이면 `UNDECIDED`
- 성능 분석: 함수 호출 구간만 10회 반복 측정 후 평균 ms 계산
- 프로그램 내부 비교는 항상 표준 라벨 `Cross`, `X` 기준으로 처리했다.
- 입력 데이터의 `+`, `cross`, `x`를 그대로 비교하지 않고 먼저 정규화한 뒤 판정했다.
